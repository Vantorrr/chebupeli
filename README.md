# Velaro - Telegram Mini App для продажи eSIM

Telegram Mini App - копия приложения Airalo с названием Velaro.

## Структура проекта

- `bot/` - Telegram бот (Node.js + Telegraf)
- `app/` - Mini App (Next.js + React + TypeScript)
- `.env` - Конфигурация (создайте на основе `.env.example`)

## Установка

### 1. Установка зависимостей

```bash
# Корневая директория
npm install

# Mini App
cd app
npm install
```

### 2. Настройка окружения

Скопируйте `.env.example` в `.env` и заполните:

```bash
cp .env.example .env
```

Обязательные переменные:
- `TELEGRAM_BOT_TOKEN` - токен бота от @BotFather
- `TELEGRAM_WEBAPP_URL` - URL вашего Mini App (например, https://your-domain.com)
- Данные для платежных систем
- API ключи провайдера eSIM (когда будут готовы)

### 3. Запуск

**Разработка:**
```bash
npm run dev
```

Это запустит:
- Telegram бот на порту 8080
- Mini App на порту 3000

**Продакшн:**
```bash
# Сборка Mini App
npm run build

# Запуск бота
npm start
```

## Функционал

### Telegram Бот
- Команда `/start` - приветствие и главное меню
- Команда `/menu` - главное меню
- Кнопки для навигации в Mini App
- Раздел поддержки и правовой информации

### Mini App

#### Главная страница
- Приветственное сообщение
- Кнопки: Тарифы, Мои eSIM, Поддержка, Правовая информация

#### Раздел "Тарифы"
- Автоматическая загрузка тарифов через API
- Фильтры: страна, регион, срок, тип пакета
- Отображение цены в рублях
- Кнопка "Купить"

#### Раздел "Оплата"
- Выбор способа оплаты (Тинькофф / ЮKassa / CloudPayments)
- Чекбокс согласия с офертой
- После оплаты - QR-код и инструкция по установке

#### Раздел "Мои eSIM"
- Список купленных пакетов
- Информация: страна, срок, оставшийся трафик
- QR-код для повторной установки

#### Раздел "Поддержка"
- Связь с поддержкой (Telegram бот)
- Email поддержки
- Быстрые решения проблем
- FAQ

#### Правовая информация
- Публичная оферта
- Политика конфиденциальности
